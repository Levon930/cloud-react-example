## Cloudike GQL Gateway

1. `.env`:
    ```
    PORT=4000
    GRAPHQL_PLAYGROUND_ENABLED=true
    LOGGER_LEVEL=dev
    CLOUDIKE_API=http://api.staging.cloudike.kr/api/1
    ```

2. `yarn`

3. `yarn start`

4. Open in browser http://localhost:3000/graphql
